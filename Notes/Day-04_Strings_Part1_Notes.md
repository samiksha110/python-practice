# 📌 What is a String?

A string is a sequence of characters enclosed in quotes.

Examples

```python
name = "Samiksha"

city = 'Pune'

course = "Python"
```

---
# 📌 Types of Quotes

## Single Quotes

```python
name = 'Python'
```

## Double Quotes

```python
name = "Python"
```

Both are exactly the same.

---

## Triple Quotes

Used for multiple lines.

```python
message = """
Hello
Welcome
Good Morning
"""
```
